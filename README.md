# 📄 CV Project using HTML & CSS

## 📌 Overview
This is a simple and responsive CV (Curriculum Vitae) project built using HTML and CSS. It showcases a clean layout and modern design suitable for personal portfolio use.

## 🎯 Features
- Clean and professional CV design  
- Structured with HTML5  
- Styled using CSS3  
- Responsive layout (mobile & desktop friendly)  
- Easy to customize  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
## 🎓 Purpose
This project is created for learning and academic purposes to practice front-end web development skills.
